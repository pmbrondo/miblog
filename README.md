Genero una web para listar y organizar las tareas de un equipo de desarrollo de varios integrantes, donde todos puedan tener acceso a los proyectos pendientes y las tareas que se deben realizar para cada uno.



Uso
Se debera ingresar a la web y generar un usuario en caso de no estar registrado. Una vez realizado tendra la posibilidad de ver los proyectos pendientes de desarrollo para el equipo y las tareas por cada uno. Se podra en caso de ser necesirio agregar nuevas tareas.
Solo el admistrador tendra la posibilidad de asignar nuevos proyectos.
